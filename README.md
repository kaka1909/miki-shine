# miki-shine

Public landing and legal pages for MikiShine apps. Served via GitHub Pages.

## Structure

```
/
├── index.html             — MikiShine landing
├── styles.css             — shared styles (dark + light)
├── .nojekyll              — disable Jekyll processing
└── smaller-images/        — Smaller Image app
    ├── privacy.html
    └── support.html
```

## Adding a new app

1. Make a new subfolder under the root (e.g. `another-app/`)
2. Copy `smaller-images/privacy.html` as a starting point
3. Link it from `index.html`

All styles are shared via the root `styles.css`.
